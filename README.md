# Proxmox VE — Deep Dive

Roadmap, Reference Cards, e Architettura Interna per Proxmox VE, basato sulla documentazione ufficiale (pve.proxmox.com/pve-docs).

Template riutilizzabile per approfondire i singoli elementi dell'infrastruttura (Sophos, switch, AD, M365...).

## Sviluppo locale
npm install
npm run dev

## Build
npm run build
